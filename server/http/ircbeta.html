Q: How Can I install the web installer
A: copy the contence of the file to a workable location like %appdata% when started it will create a ygopro folder. Then copy your DevPro or YGOPro scripts pics and decks to the folder.

Q: I can not save a deck how can I correct this?
A: Create a deck folder in the /ygopro/ folder of the installation.
